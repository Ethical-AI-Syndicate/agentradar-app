Authentication

AgentRadar uses JWT-based authentication to secure your account and data. All API requests must include a valid bearer token.

Getting Your API Token

POST https://api.agentradar.app/auth/login
Content-Type: application/json

{
"email": "your-email@example.com",
"password": "your-password"
}

Using Your Token

GET https://api.agentradar.app/alerts
Authorization: Bearer YOUR_JWT_TOKEN
